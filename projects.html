What this page is

This page is a simple overview of projects and practice work. I am keeping it organized and focused, because the goal is to show what I have built and what I learned from it, not to overwhelm the page with random details.

Right now I am practicing controlled CSS changes, so the site stays readable while I learn how selectors target semantic structure.

For now, the structure has to carry the page. Later, once we cover spacing, layout, and typography, I will refine how it looks without losing the meaning.

Project highlights

Semantic HTML three page site

This project is the website you are currently on. The goal is to build three pages with real content using semantic HTML elements and a clear heading outline. I am intentionally keeping styling light so the structure stays the focus.

The biggest improvement I have made through this project is learning to think about content as part of the design. If the content is weak or confusing, the site fails even if it looks nice.

Basic PHP practice

I have also been building simple PHP scripts that receive user input, validate it, and print results. This kind of work connects to backend fundamentals because it forces you to be careful with user data and conditions.

I like this practice because it is close to how real apps work. A lot of software is just input, validation, and predictable output.
