What this site is for

This website is a course project focused on writing real content and organizing it using semantic HTML5. The point is to build pages that make sense even without styling, so the structure and meaning come first.

I am using headings to create a clear outline on each page. If someone skimmed only the headings, they should still understand what the page is about and where to find information.

Instead of filling the page with extra features, I am focusing on basic web fundamentals that support usability. This includes readable paragraph content, consistent navigation, and predictable page organization.

My background with HTML

Hands on experience

I have used HTML in real work settings, mainly to build and edit web forms and structured content for users. That experience taught me that small details matter, like clear labels, good ordering, and content that matches what the user expects.

What I am practicing in this project

In this project I am practicing semantic elements such as header, nav, main, section, article, and footer. I want the page to communicate meaning through structure instead of relying on layout tricks.
